--- layout: post title: "Acessando a API do Spotify" subtitle: "Um tutorial rápido para liberar a pessoa analista de dados que vivem em você." date: 2020-05-20 15:00:00 -0300 background: '' --- Opa, meus anjos!

Se você chegou até aqui provavelmente está querendo mexer com dados do Spotify. Primeiro você tem que conseguir ter acesso à API e por isso cá estamos!

Primeiro de tudo, este é o Spotify for Developers. Tudo sobre como o Spotify funciona por trás das câmeras está aqui.

Na sessão chamada discover onde você encontra o que a comunidade andou fazendo com o acesso aos dados. Quem sabe não te inspira?

No guia geral estão as informações sobre como configurar sua aplicação e como você deve usar a API. Vale a pena dar uma lida. Mas vamos ao nosso rolê.

  1. Entre na dashboard do Spotify for Developers aqui e faça login com sua conta no Spotify.
  2. Login feito? Agora clicando em "create client id" você fará a requisição de acesso à API
  3. E a partir daqui você vai responder algumas perguntas sobre o seu projeto.
  4. Se você quer apenas se aventurar na API, lembre de colocar que sua aplicação é não comercial.
  5. Se tudo deu certo, é isso que vai aparecer para você:
  6. Tudo pronto. Agora só clicar em "show client secret" e você terá o seu id e senha para a conexão com a API. Daqui para frente é uso e sucesso.

Considerações legais:

  • O Spotify coloca alguns dados da sua aplicação em visualizações. Você pode ver o uso da sua aplicação, quantos usuários a acessam e outras coisas. Gostasse?
  • Claro que como PyLady super incentivo o uso do Spotipy para mexer com a API. Não é tão complicado depois que você entende os comandos e quem sabe não teremos um tutorial escrito sobre isso depois?
